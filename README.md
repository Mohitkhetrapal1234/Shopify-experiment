# Premium Ethnic Indian Womenswear Shopify Store

This repo is the local working directory for a test Shopify build focused on premium ethnic Indian womenswear.

## Goals

- Set up a Git-backed Shopify theme development workflow
- Connect the project to a Shopify development store with Shopify CLI
- Build a high-end storefront for sarees, lehengas, kurtas, and occasionwear

## Suggested brand direction

- Positioning: modern luxury rooted in Indian craft
- Target audience: women shopping for festive, wedding, and occasionwear
- Visual tone: rich editorial imagery, generous whitespace, high-contrast typography, gold or jewel-tone accents
- Merchandising priorities: collections by occasion, fabric, craft, silhouette, and wedding edit

## Local workflow

1. Install dependencies with `npm install`
2. Authenticate with your Shopify partner or store account
3. Run `npm run dev`
4. Commit changes to Git and push to GitHub

## First build checklist

- Create a Shopify dev store
- Decide the brand name, logo direction, and color system
- Prepare hero imagery and 8-12 initial products
- Configure collections, navigation, policies, and shipping rules
- Customize homepage, PDP, and collection templates

## Project structure

- `theme/`: Shopify theme code
- `STORE_STRATEGY.md`: merchandising and UX direction
- `SETUP.md`: GitHub and Shopify connection steps
